# Mini ERP

*Proyecto final de la materia Tecnologías Emergentes de la Programación.*

## Stack

[Vite](https://vite.dev/)
[TypeScript](https://www.typescriptlang.org/)
[Tailwind CSS](https://tailwindcss.com/)
[shadcn](https://ui.shadcn.com/)
[Mini ERP API](https://github.com/Piuliss/mini-erp)

## Usar el proyecto

Copiar el repositorio

```sh
git clone https://github.com/facu18xk/reactFinal
cd reactFinal
```
Instalar dependencias

```sh
pnpm install
```
Inciiar el servidor de desarrollo

```sh
pnpm run dev
```
